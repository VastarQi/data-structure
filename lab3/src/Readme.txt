This is the application of Huffman coding

Version: Java 8;
SDK: 16 Oracle OpenJDK Version 16.0.1;
IDE: Intellij;



Initiate the conversion by typing your I/O path like:

java Compression C:\...\input.txt C:\...Output.txt

file Compression is for cleartext to huffmancode

file Decompression is for  huffmancode to cleartext 

Use PrintHuffmanCodeTable & PrintTreePreorder to print the code table and tree

Check your input format before running the program
